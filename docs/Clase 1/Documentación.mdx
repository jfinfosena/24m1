## Sintaxis Básica para Arduino:

**1. Variables:**

* **Declaración:** Se declara una variable con su tipo de dato y su nombre, seguido de un punto y coma.
  ```cpp
  int edad; // Declaración de una variable entera llamada "edad"
  float altura; // Declaración de una variable de punto flotante llamada "altura"
  char inicial; // Declaración de una variable de carácter llamada "inicial"
  bool encendido; // Declaración de una variable booleana llamada "encendido"
  ```
* **Tipos de Datos:**
    * **int:** Entero (ej: 10, -5, 0)
    * **float:** Número decimal (ej: 3.14, -2.5)
    * **char:** Caracter (ej: 'A', '!', '?')
    * **bool:** Booleano (Verdadero/Falso) (ej: true, false)
* **Asignación de Valores:** Se le asigna un valor a una variable con el operador `=`.
  ```cpp
  edad = 25; // Asignando el valor 25 a la variable "edad"
  altura = 1.75; // Asignando el valor 1.75 a la variable "altura"
  inicial = 'J'; // Asignando el caracter 'J' a la variable "inicial"
  encendido = true; // Asignando el valor "verdadero" a la variable "encendido"
  ```

**2. Operadores:**

* **Aritméticos:**
    * `+` Suma
    * `-` Resta
    * `*` Multiplicación
    * `/` División
    * `%` Módulo (resto de la división)
  ```cpp
  int suma = 5 + 3; // Suma 5 y 3, guarda el resultado en "suma"
  int resta = 10 - 4; // Resta 4 de 10, guarda el resultado en "resta"
  int producto = 7 * 2; // Multiplica 7 por 2, guarda el resultado en "producto"
  int cociente = 15 / 3; // Divide 15 entre 3, guarda el resultado en "cociente"
  int resto = 17 % 5; // Divide 17 entre 5, guarda el resto en "resto"
  ```
* **Relacionales:**
    * `==` Igual a
    * `!=` Diferente de
    * `>` Mayor que
    * `<` Menor que
    * `>=` Mayor o igual que
    * `<=` Menor o igual que
  ```cpp
  bool sonIguales = 5 == 5; // Comprueba si 5 es igual a 5, guarda el resultado en "sonIguales"
  bool esMayor = 10 > 7; // Comprueba si 10 es mayor que 7, guarda el resultado en "esMayor"
  bool esMenor = 3 < 8; // Comprueba si 3 es menor que 8, guarda el resultado en "esMenor"
  ```
* **Lógicos:**
    * `&&` Y lógico
    * `||` O lógico
    * `!` Negación
  ```cpp
  bool esVerdadero = true && true; // Comprueba si ambas expresiones son verdaderas, guarda el resultado en "esVerdadero"
  bool esFalso = false || false; // Comprueba si al menos una expresión es verdadera, guarda el resultado en "esFalso"
  bool negacion = !true; // Niega la expresión true, guarda el resultado en "negacion"
  ```
* **Asignación:**
    * `=` Asignación
    * `+=` Suma y asignación (a += b es equivalente a a = a + b)
    * `-=` Resta y asignación
    * `*=` Multiplicación y asignación
    * `/=` División y asignación
    * `%=` Módulo y asignación
  ```cpp
  int numero = 10; // Asigna el valor 10 a la variable "numero"
  numero += 5; // Suma 5 al valor actual de "numero"
  numero -= 2; // Resta 2 al valor actual de "numero"
  ```

**3. Instrucciones de Control:**

* **Condicionales (if, else if, else):**
    * `if` : Ejecuta un bloque de código si una condición es verdadera.
    * `else if` : Ejecuta un bloque de código si la condición anterior es falsa y la actual es verdadera.
    * `else`: Ejecuta un bloque de código si ninguna de las condiciones anteriores es verdadera.
  ```cpp
  int edad = 18;
  if (edad >= 18) {
    Serial.println("Eres mayor de edad");
  } else {
    Serial.println("Eres menor de edad");
  }
  ```
* **Bucles:**
    * **for:** Ejecuta un bloque de código un número determinado de veces.
      ```cpp
      for (int i = 0; i < 5; i++) {
        Serial.println(i); // Imprime los números del 0 al 4
      }
      ```
    * **while:** Ejecuta un bloque de código mientras una condición sea verdadera.
      ```cpp
      int contador = 0;
      while (contador < 10) {
        Serial.println(contador); // Imprime los números del 0 al 9
        contador++; // Incrementa el contador en 1
      }
      ```
    * **do while:** Ejecuta un bloque de código al menos una vez, y luego repetidamente mientras una condición sea verdadera.
      ```cpp
      int contador = 0;
      do {
        Serial.println(contador); // Imprime los números del 0 al 9
        contador++; // Incrementa el contador en 1
      } while (contador < 10);
      ```

**4. Funciones:**

* **Declaración:**
  ```cpp
  int suma(int a, int b) {
    return a + b;
  }
  ```
  * **Nombre:** "suma" (debes elegir un nombre descriptivo)
  * **Tipo de retorno:** "int" (indica el tipo de dato que devuelve la función)
  * **Parámetros:** "int a", "int b" (variables que recibe la función)
  * **Cuerpo de la función:** Código que se ejecuta al llamar a la función.
  * **Instrucción return:** Devuelve un valor del tipo especificado.
* **Llamada:**
  ```cpp
  int resultado = suma(5, 3); // Llama a la función "suma" con los argumentos 5 y 3
  Serial.println(resultado); // Imprime el resultado de la suma (8)
  ```

**Ejemplos en Arduino:**

* **Control de un LED con un Botón:**
  ```cpp
  int ledPin = 13; // Define el pin del LED
  int buttonPin = 2; // Define el pin del botón
  int estadoLed = LOW; // Estado inicial del LED (apagado)

  void setup() {
    pinMode(ledPin, OUTPUT); // Configura el pin del LED como salida
    pinMode(buttonPin, INPUT); // Configura el pin del botón como entrada
  }

  void loop() {
    int valorBoton = digitalRead(buttonPin); // Lee el estado del botón
    if (valorBoton == HIGH) { // Si el botón está presionado
      estadoLed = !estadoLed; // Cambia el estado del LED
      digitalWrite(ledPin, estadoLed); // Enciende o apaga el LED
      delay(500); // Espera 500ms para evitar rebotes del botón
    }
  }
  ```
* **Lectura de un Sensor de Temperatura:**
  ```cpp
  int sensorPin = A0; // Define el pin del sensor de temperatura
  float temperatura; // Define la variable para almacenar la temperatura

  void setup() {
    Serial.begin(9600); // Inicializa la comunicación serial
  }

  void loop() {
    int valorSensor = analogRead(sensorPin); // Lee el valor del sensor
    temperatura = (valorSensor * 5.0) / 1023.0; // Convierte el valor analógico a voltaje
    Serial.print("Temperatura: ");
    Serial.println(temperatura); // Imprime la temperatura en la consola serial
    delay(1000); // Espera 1 segundo
  }
  ```
