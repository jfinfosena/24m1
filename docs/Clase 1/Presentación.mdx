# Introducción

## ¿Qué es Arduino?

Arduino es una plataforma de hardware y software de código abierto que facilita la creación de proyectos electrónicos interactivos. Se compone de dos elementos principales:

### **1. Hardware:**


* **Placas Arduino:** Son placas de desarrollo con un microcontrolador, entradas y salidas digitales y analógicas, interfaces de comunicación (como USB, SPI, I2C) y otros componentes esenciales. Existen diferentes modelos, cada uno con características y funcionalidades específicas.
    * **Arduino Uno:**  Modelo básico y popular, ideal para principiantes.
    * **Arduino Mega:** Ofrece más pines de E/S y memoria para proyectos más complejos.
    * **Arduino Nano:** Placa pequeña y versátil, perfecta para proyectos portátiles.
    * **Arduino Due:**  Utiliza un microcontrolador ARM de mayor potencia.
    * **Arduino Leonardo:** Ofrece un mayor control sobre el puerto USB.
* **Componentes adicionales:**  Se complementan con las placas Arduino para crear proyectos más completos.
    * **Sensores:** Detectan información del entorno (luz, temperatura, movimiento, etc.).
    * **Actuadores:** Controlan dispositivos (motores, LEDs, relés, etc.).
    * **Pantallas:** Permiten mostrar información o interactuar con el usuario.
    * **Módulos de comunicación:** Facilitan la comunicación inalámbrica (Bluetooth, WiFi).

### **2. Software:**

* **Entorno de desarrollo Arduino (IDE):** Se trata de un software de código abierto que se utiliza para escribir, compilar y cargar programas (sketch) a las placas Arduino. Es fácil de usar,  incluye ejemplos predefinidos y facilita la depuración del código.

**Historia y Evolución de Arduino:**

* **2005:** Nacido en el Instituto de Diseño Interactivo de Ivrea (Italia), Arduino se creó como una herramienta para estudiantes de diseño y arte que buscaban crear proyectos interactivos.
* **2008:**  Se lanza la placa Arduino Diecimila, que se convierte en el estándar de la plataforma.
* **2010:** Se publica la versión 1.0 del IDE, que se ha convertido en el software de desarrollo más utilizado.
* **2012:** Se lanza la placa Arduino Uno, con un diseño más actualizado y mejorado.
* **Actualidad:** La plataforma Arduino ha evolucionado constantemente, incorporando nuevos modelos de placas, funcionalidades y herramientas.

**Ventajas de Usar Arduino:**

* **Fácil de usar:**  Tanto el hardware como el software están diseñados para ser accesibles a personas sin experiencia previa en electrónica.
* **De código abierto:**  El diseño de hardware y software es libre, permitiendo a los usuarios modificarlos y adaptarlos a sus necesidades.
* **Gran comunidad:**  Una vasta comunidad global de usuarios comparte recursos, proyectos y ayuda a resolver problemas.
* **Versatilidad:**  Se puede utilizar para una gran variedad de proyectos, desde simples prototipos hasta aplicaciones más complejas.
* **Bajo costo:**  Las placas Arduino y los componentes son relativamente económicos, lo que lo convierte en una opción accesible para aficionados y profesionales.
